SVOGV is an ongoing project that helps Angular developers to create awesome apps quite fast.

Objections

This project consists of several parts that can be used independently:

  1. A validation solution to automate form creation, based on decorators and editors
  2. A widget library for form components that use Bootstrap 4 as layout foundation
  3. A widget library based on SVG with some very special components

What's up in the future?

I'm going to treat this project as a starter for learning and experimenting.

  • A base path is called @svogv
  • All design goes to Bootstrap 4
