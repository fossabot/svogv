SVOGV is a ongoing project that helps Angular2 developers to create awsome apps quite fast.

Objections

This project consists of several parts that can be used independently:

  1. A validation solution to automate form creation, based on decorators and editors
  2. A widget library for components that use Bootstrap 4 as layout foundation
  3. A widgert library based on SVG with some very special components
  4. A HUD widget library for buildung sci-fi designs, mostly SVG based

So, stay tuned!
